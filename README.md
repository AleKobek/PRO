# PL

Temat pracy: **"System Pozwalający Graczom Zrzeszać Się i Dzielić Pasją"**

Nazwa aplikacji: **"Squadra"**

Technologie:
- frontend - JavaScript wraz z React
- backend - C# wraz z ASP.NET i Entity Framework
- baza danych - MS SQL


W swojej końcowej formie, aplikacja będzie pozwalała użytkownikom:
- założyć konto
- dostosować swój profil o dane takie jak pseudonim, znane języki oraz biegłość w nich, opis, awatar
- ustawić status, który ma się wyświetlać innym użytkownikom, gdy jest online
- pobierać dane o swoich statystykach w grach wideo z głównych platform gamingowych
- tworzyć "drużyny", do których może dołączyć kilku użytkowników, oraz ustawić im wymagania na podstawie statystyk
- tworzyć "gildie", do których może dołączyć kilkudziesięciu użytkowników oraz ustawić im wymagania na podstawie statystyk.
  - każda gildia będzie miała dodatkowo własny czat i tabelę wyników
- dodawać znajomych i prowadzić z nimi czat prywatny

# EN
Engineering thesis title: **"A System Allowing Gamers to Gather and Share Their Passion"**

Application Name: **"Squadra"**

Technologies:
- frontend - JavaScript with React
- backend - C# with ASP.NET and Entity Framework
- database - MS SQL

In its final form, the application will allow users to:
- create an account
- customize their profile with data such as a nickname, languages ​​spoken and proficiency, description, and avatar
- set a status to be displayed to other users when online
- download their video game stats from major gaming platforms
- create "teams" that multiple users can join and set requirements based on their stats
- create "guilds" that dozens of users can join and set requirements based on their stats.
  - each guild will also have its own chat and leaderboard
- add friends and chat privately with them
